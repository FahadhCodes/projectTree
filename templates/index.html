Decision
Tree Classifier

A Decision Tree Classifier is a supervised learning algorithm that splits the feature space into hierarchical regions using a sequence of binary decisions. Each internal node tests a feature against a threshold; leaves assign a class label. Splits are chosen to maximise Information Gain (or minimise Gini Impurity), progressively reducing the entropy of the training set until pure or near-pure partitions are reached.

This visualiser walks through the ID3/CART algorithm step-by-step, exposing entropy at each node, feature importances, and the final class distribution across leaf nodes.
